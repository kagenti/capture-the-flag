# kagenti/capture-the-flag 
Games of capture the flag, meant to demonstrate and test Kagenti's identity and access control systems.

This experiment was inspired by a talk by Nicholas Carlini from Anthropic, where current generation frontier models were found to be exceedly capable at zero-day exploitation. Video [here](https://www.youtube.com/watch?v=1sd26pWhfmg).

Additional prior-art may also be found here via the ["Agents of Chaos"](https://agentsofchaos.baulab.info/report.html) study.

[Kagenti](https://github.com/kagenti/kagenti) is a project meant to help secure agents in production systems.

Ported from an initial experiment by [usize](https://github.com/usize/kagenti-ctf).
